# Proyecto: Aplicador de Filtro Bilateral a Imágenes

Este proyecto consiste en un programa que aplica un filtro bilateral a imágenes en escala de grises. Fue desarrollado como parte de una tarea de la asignatura **Sistemas Lineales y Señales** en la universidad de Cuenca.

## Objetivo del Proyecto

El objetivo de este proyecto es implementar y aplicar el filtro bilateral a imágenes en escala de grises, usando el conocimeinto aprendido e investigado en la materia **Sistemas Lineales y Señales**.

## Descripción

Este proyecto tiene como objetivo aplicar un filtro bilateral a imágenes en escala de grises, utilizando dos enfoques: 

1. **Implementación propia**: Los estudiantes desarrollarán un código que aplica el filtro bilateral manualmente, con el fin de comprender mejor los principios matemáticos y algoritmos subyacentes.

2. **Uso de bibliotecas especializadas**: También se aplicará el filtro bilateral utilizando librerías dedicadas (como OpenCV), que implementan de manera optimizada este tipo de procesamiento. 

El propósito de utilizar ambos enfoques es comparar los resultados, evaluando las diferencias en precisión y rendimiento entre el código desarrollado por los estudiantes y la implementación optimizada de las bibliotecas.

## Contenido del Proyecto

El proyecto incluye:

- **Código fuente**: Implementación en Python del filtro bilateral para imágenes en escala de grises.
- **Imágenes de prueba**: Algunas imágenes de ejemplo en escala de grises para probar el filtro.
- **Instrucciones de uso**: Explicación de cómo ejecutar el código y aplicar el filtro a las imágenes.
